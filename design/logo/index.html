Concept 1 — The Explicit Route

The idea

●───◆───● — two explicit endpoints (circles) connected through a central routing exchange (diamond). Every node and every connection is drawn on purpose. That is BMF: you declare the topology, nothing is hidden.

The primary mark folds this route into an M: the message enters at the left endpoint, climbs, is routed through the diamond at the central junction, and descends to the right endpoint.

Tune it live

Adjust the base line thickness and the diamond size and watch every size update in real time — handy for checking how the mark holds up at favicon sizes.

Defaults: variant A · base thickness 8.0 · diamond size 17.0

Variant

Variant A · vertex junction

The diagonals meet at the diamond's top vertex — one clean peak.

Variant B · face junction

The diagonals dock into the middle of the diamond's upper-left and upper-right edges, leaving the exchange's peak visible between the incoming routes.

Variant C · raised exchange

Variant B with the diamond lifted up so it is no longer centered between the two vertical strokes. The diagonals drop less, so the M's middle lines read flatter.
